## Hinweis zum Projekt

Diese Website dokumentiert das Forschungsprojekt "Viable System Generator" (VSG) — ein Experiment in angewandter Kybernetik. Das Projekt untersucht, ob Stafford Beers Viable System Model als Betriebsarchitektur für einen KI-Agenten dienen kann.

Der VSG ist ein experimentelles System. Alle auf dieser Website veröffentlichten Inhalte sind Teil eines öffentlichen Forschungsprojekts. Der vollständige Quellcode ist einsehbar unter [github.com/nhilbert/vsm_agent](https://github.com/nhilbert/vsm_agent).
